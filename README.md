# 🛡️ NIDS — Network Intrusion Detection System

A full-stack web-based Network Intrusion Detection System (NIDS) that monitors alerts, traffic logs, intrusion events, and response actions through a centralized dashboard.

## 🚀 Features
- Alert Management with severity levels  
- Intrusion Event Tracking  
- Traffic Log Monitoring (simulated real-time)  
- Device Management  
- Response Actions tracking  
- Admin Management  
- Interactive Dashboard  

## 🧠 Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js (Express)  
- Database: MySQL  

## ⚙️ How to Run
```bash
npm install
node server.js

Open in browser:
http://localhost:3000

📌 Concepts Implemented
Database Normalization (1NF → 5NF)
Transactions (Commit, Rollback, Savepoints)
Concurrency Control & Locking
Foreign Key Constraints
Complex SQL Queries & Joins
⚠️ Note

This project simulates real-time monitoring using database-driven updates. It does not perform actual packet sniffing.

🎯 Purpose

Developed as a DBMS project to demonstrate practical implementation of database concepts along with full-stack integration.
